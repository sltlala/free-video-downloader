# 万能视频下载网站 - 方案设计文档

## 1. 技术架构

```
┌──────────────────────────────────────────────────────────┐
│                     用户浏览器                            │
│  ┌─────────────────────────────────────────────────────┐ │
│  │           Vue3 + Vite + Tailwind CSS               │ │
│  │  URL输入 → 解析展示 → 格式选择 → 下载              │ │
│  └──────────────────────┬──────────────────────────────┘ │
└─────────────────────────┼────────────────────────────────┘
                          │ HTTP API
┌─────────────────────────┼────────────────────────────────┐
│                  FastAPI 后端                              │
│  ┌──────────────────────┴──────────────────────────────┐ │
│  │  /api/parse     → 解析视频信息                       │ │
│  │  /api/download  → 服务端下载并提供文件               │ │
│  │  /api/direct-url → 获取视频直链                      │ │
│  └──────────────────────┬──────────────────────────────┘ │
│                         │                                 │
│  ┌──────────────────────┴──────────────────────────────┐ │
│  │              yt-dlp 封装层                           │ │
│  │  extract_info() → 解析视频元数据                     │ │
│  │  download()     → 下载视频文件                       │ │
│  │  get_url()      → 提取直链地址                       │ │
│  └────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────┘
```

## 2. 技术栈选型

| 层次 | 技术 | 选型理由 |
|------|------|----------|
| 前端框架 | Vue 3 + Vite | 轻量、上手快、生态成熟 |
| CSS 框架 | Tailwind CSS | 快速构建精美 UI，无需写大量自定义 CSS |
| 后端框架 | FastAPI | 异步高性能、自带 API 文档、Python 生态 |
| 核心能力 | yt-dlp | 14w+ Star，支持 1800+ 网站，社区活跃 |
| 数据库 | 无 | 轻量设计，无需持久化存储 |

## 3. API 设计

### 3.1 解析视频信息

```
POST /api/parse
Content-Type: application/json

Request:
{
    "url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ"
}

Response (200):
{
    "success": true,
    "data": {
        "id": "dQw4w9WgXcQ",
        "title": "视频标题",
        "thumbnail": "https://i.ytimg.com/vi/xxx/maxresdefault.jpg",
        "duration": 212,
        "duration_string": "3:32",
        "uploader": "上传者名称",
        "platform": "YouTube",
        "view_count": 1500000,
        "upload_date": "20240101",
        "formats": [
            {
                "format_id": "137+140",
                "ext": "mp4",
                "resolution": "1920x1080",
                "height": 1080,
                "filesize": 52428800,
                "filesize_approx": 52428800,

                "acodec": "mp4a",

```

### 3.2 下载视频（服务端代理模式）

```
POST /api/download
Content-Type: application/json

Request:
{
    "url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ",
    "format_id": "137+140"
}

Response: 视频文件流 (Content-Type: video/mp4)
Headers:
  Content-Disposition: attachment; filename="视频标题.mp4"
```

### 3.3 获取直链

```
POST /api/direct-url
Content-Type: application/json

Request:
{
    "url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ",
    "format_id": "137"
}

Response (200):
{
    "success": true,
    "data": {
        "direct_url": "https://rr3---sn-xxx.googlevideo.com/...",
        "ext": "mp4",
        "filesize": 52428800
    }
}
```

## 4. 下载模式策略

系统自动选择最优下载模式：

1. **优先尝试直链模式**：调用 `/api/direct-url` 获取直链
   - 如果成功且直链可用 → 浏览器直接下载（不占服务器带宽）
2. **回退到代理模式**：如果直链不可用或有防盗链
   - 调用 `/api/download` → 服务端下载后流式传输给用户

## 5. 前端页面设计

### UI 风格参考
参考 ai.codefather.cn/painting 的设计风格：
- 现代简约，蓝白主色调
- 主色 `#1777FF`，背景白色 `#FFFFFF`，浅灰区域 `#F8FAFC`
- 胶囊形按钮（border-radius: 9999px）
- 12px 圆角卡片 + 轻阴影
- 响应式设计，移动端适配

### 页面结构
1. **Header**：Logo + 导航
2. **Hero 区域**：大标题 + 副标题 + URL 输入框
3. **视频信息区**（解析成功后显示）：缩略图 + 信息 + 格式选择 + 下载按钮
4. **功能亮点区**：4 个特性卡片
5. **VIP 套餐区**：免费版 vs 高级版
6. **平台展示区**：支持的平台 Logo
7. **Footer**：版权和免责声明
